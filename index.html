脚本筛选 / VCP

vcp_plot.py：把候选股变成可复核图表

这个模块服务于“先筛，再看图”的实战流程：筛选器先压缩股票池， vcp_plot.py 再读取最新 vcp_result_*.csv， 拉取前复权日线并生成带 K 线、均线、52 周高点和成交量的复盘页面。

输入

承接筛选结果

自动读取当前目录最新的 vcp_result_*.csv， 不在画图阶段重新全市场扫描。

复盘

图形验证形态

每只候选股生成交互图，重点观察趋势、回撤、缩量、振幅收缩和是否接近 52 周高位。

边界

脚本不替代判断

规则只负责缩小观察范围；最终仍要人工确认市场环境、板块位置、流动性和买卖点。

命令

本地离线生成

运行 python vcp_screener.py 后接 python vcp_plot.py，再把静态结果发布到网站。
