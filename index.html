脚本筛选 / 完整说明

候选名单如何产生，又如何被图形复核

准确地说，筛选条件写在 vcp_screener.py 中； vcp_plot.py 不重新全市场筛选，而是把命中的股票做成可交互的日线复盘页。 两个脚本连在一起，才构成当前使用的 VCP 观察流程。

01 / Workflow

脚本流程

1

筛选器先生成名单

vcp_screener.py 扫描沪深主板，按趋势、回撤、量能与振幅收缩条件判断， 将通过者保存为 vcp_result_YYYYMMDD.csv。

2

绘图脚本读取最新结果

find_latest_csv() 在当前目录寻找最新的 vcp_result_*.csv。画图阶段只复核已命中候选，不重复扫描市场。

3

拉取前复权日线

通过 AkShare 的新浪财经源获取行情，代码转换为 sh 或 sz 格式；默认 8 个线程并发，每只股票最多重试 4 次。

4

生成交互复盘页面

每只候选股输出 Plotly 日线图，最终生成 vcp_charts_YYYYMMDD_HHMM.html： 顶部为汇总表，下面逐只查看形态。

02 / Rules

筛选规则

以下条件来自当前使用中的 vcp_screener.py。

股票池

限定沪深主板

只保留代码以 60 或 00 开头的股票， 并剔除名称含 ST 或退市标记的样本。

周线趋势

大级别必须向上

最新周线满足 Close > MA10 > MA30， 且周 MA10、MA30 相比 4 周前均在上升。

回撤位置

回调但不破趋势

收盘价高于日线 MA50；距 52 周高点回撤位于 (3%, 25%]；且高于 52 周低点的 1.3 倍。

量能收缩

近 5 日明显缩量

要求 vol5 / vol20 < 0.8： 最近 5 日平均成交量低于最近 20 日均量的 80%。

波动收缩

振幅同步收窄

日振幅按 (high - low) / pre_close 计算， 并要求 amp5 / amp20 < 0.8。

阶梯收缩

尾端更安静

将最近 5 日拆成前 2 日与后 3 日，要求后 3 日平均振幅更低， 尝试捕捉整理尾端继续变紧的状态。

03 / Output

图表输出与使用方式

一页完成列表与图形复核

图表标题展示代码、名称、收盘价、回撤比例、vol5/vol20 与 amp5/amp20。主图包含 K 线、MA10、MA20、MA50 和 52 周高点虚线，副图显示红涨绿跌的成交量柱。

脚本会去掉非交易日空隙；单只股票拉取失败时将其记录并跳过， 不妨碍其余候选继续生成复盘图。

04 / Command

python vcp_screener.py
python vcp_plot.py

第一行负责筛选与保存候选 CSV，第二行读取最新结果并输出交互图表。 规则负责收窄观察范围，最终的买卖判断仍需要结合图形、环境和交易计划。

S1 2022.08 下跌中的收益

2022-08-13 至 2022-08-29，BTC 约 -17.0%，资产反而增加约 268.5 万 USDT。资料推断：可能具备做空能力，不是单边多头。

L1 2024.11 突破段

2024-11-06 至 2024-11-22，BTC 约 +30.9%，资产增加约 556.8 万 USDT。更像是在突破行情中押对方向并放大仓位。

L2 2024.11-12 高弹性延续

2024-11-22 至 2024-12-08，BTC 仅约 +2.3%，资产增加约 617.1 万 USDT。资料将其归为高杠杆或仓位集中带来的高弹性。

L3 2025.05 小涨大赚

2025-05-17 至 2025-06-02，BTC 约 +2.7%，资产增加约 478.9 万 USDT。小幅行情里出现大额资产跳升，继续指向高杠杆或集中仓位。

L4 2025.07 最大盈利跳升

2025-07-04 至 2025-07-20，BTC 约 +8.6%，资产增加约 1253.6 万 USDT，并触及约 2457.7 万 USDT 峰值。

L5 2026.01 再次快速增长

2025-12-27 至 2026-01-12，BTC 约 +3.9%，资产增加约 740.7 万 USDT。资料推断：BTC 小涨但资产大幅增加，可能来自高杠杆或仓位集中。

S2 2026.03 下跌区间获利

2026-03-17 至 2026-04-02，BTC 约 -9.5%，资产增加约 344.1 万 USDT。该节点保留为空头盈利样本。

S3 2026.05 下跌区间获利

2026-05-04 至 2026-05-20，BTC 约 -2.9%，资产增加约 286.0 万 USDT。该节点继续支持“会切换多空方向”的推断。
